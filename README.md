# CompassReClasser
Adjust the html class (style) to change the colors to something that doesnt hurt your eyes or look like vomit, as well as making it easy to distinguish different classes at a glance.

To install it you will need to unzip the file, then go to chrome://extensions or edge://extensions depending on your browser, then press "Developer Mode", and then "Load Unpacked" and select the unzipped folder. Then boom! Go to compass, press reload then press the extension's icon to choose your colours.
